# react-cnode
基于webpack + react + react-router + redux + less + flex-css-layout + ES6 的React版cnode社区

### 安装
```
  npm react-kelink 
  npm install (安装依赖模块)
  npm webpack -g (没有安装webpack的需要安装)
```

### 运行
```
  npm webpack -p (编译成生产环境)
  npm start (启动服务器，访问：http://localhost:3000/)
  
```
